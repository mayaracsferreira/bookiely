# Bookiely

## Description
Library management system created for study purposes on [Nest](https://github.com/nestjs/nest) and [GraphQL](https://graphql.org/graphql-js/)

### Functionalities

- Add book in catalog (POST)
- Borrow book (PUT)
- Display available books (GET)
- Return book (PUT)
- Remove boom from catalog (DELETE)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
